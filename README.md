# Swipe Courier (Mobile)

Mobile arcade portfolio project (vertical slice). Short sessions, swipe controls, deliveries, upgrades, and performance-friendly visuals.

## Goals
- Demonstrate mobile gameplay loop design
- Touch input (swipe), UI/UX, optimization (pools, batching)
- Clean project structure and readable code

## Skills Demonstrated
- Mobile gameplay loop design (short sessions, retention-oriented flow)
- Touch controls with Unity Input System (swipe gestures)
- UI implementation (HUD, pause, results, settings)
- Progression systems (upgrades, missions)
- Performance optimization for mobile (object pooling, reduced allocations)
- Production hygiene: consistent project structure, readable code, versioning

## Core Gameplay
- Swipe left/right to switch lanes
- Avoid obstacles, collect orders
- Deliver within the timer → earn rewards → upgrade scooter

## Planned Features
- Level blocks / procedural road segments
- Upgrade system (speed, handling, capacity)
- Missions & progression
- Localization (EN/RU)

## Tech
- Unity 2022.3 LTS
- URP, TextMeshPro
- Input System
- Unity Localization

## How to Run
1. Open with Unity 2022.3 LTS
2. Load scene: `Boot` → `MainMenu` → `Gameplay`
